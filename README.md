# SQL2HTML
Cette commande, écrit en Pascal (Turbo Pascal ou Free Pascal), permet de convertir un fichier SQL en HTML.

<h3>Syntaxe</h3>

<b>SQL2HTML</b> source.SQL

<h3>Paramètres</h3>

<table>
  <tr>
    <th>Nom</th>
    <th>Description</th>
  </tr>
  <tr>
    <td><i>fichier</i></td>
    <td>Ce paramètre permet d'indiquer le fichier du code source SQL.</td>
  </tr>
</table>
